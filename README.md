# objlab, Simple Wavefront .obj viewer + some featues

`objlab` is a simple Wavefront .obj viewer + some features, written in C++11.

## Requirements

* cmake
* OpenGL 3.x

## Build

### Setup

```
$ git submodule update --init --recursive
```

### CMake build

```
$ mkdir build
$ cd build
$ cmake ..
$ make
```
